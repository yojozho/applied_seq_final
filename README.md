## ASI Final Project

This repository includes all code written for RNA-seq analysis and all generated figures.

- `de_analysis.rmd`: main file. includes all code written for PCA and differential expression analysis
- `transcripts_to_genes.R`: used to convert kallisto transcription counts to gene counts
- `make_counts_file.py`: generates all counts files for downstream analysis
- `bin` folder: includes all scripts written for data download, quality assessment, and alignment.
- `figures` folder: includes all figures from the final writeup
- other folders were generated as a part of analysis and include outputs from DESeq2, edgeR, and Sleuth
